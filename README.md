# GestãoPro

  Software de gestão comercial compliant com a legislação portuguesa.

  ## Módulos
  - Vendas (FT, FS, FR, NC, ND)
  - Compras
  - Inventário
  - Clientes e Fornecedores
  - Contas Correntes
  - Bancos
  - Mapas de Exploração
  - Exportação SAF-T (Portaria 302/2016)

  ## Tecnologias
  - Frontend: React + TypeScript + Vite + Shadcn/UI + TailwindCSS
  - Backend: Express + TypeScript
  - Base de Dados: PostgreSQL + Drizzle ORM
